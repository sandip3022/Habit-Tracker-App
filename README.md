# Habit-Tracker-App
A Flutter Habit Tracker app built with Clean Architecture, Riverpod, and Hive.


## Vibe: Clean, calm, and focused.

## Typography: Serif for Headlines (Classy) + Sans-Serif for Body (Readable).

## Palette:

### Primary: Midnight Navy (Professional, Focus)

### Secondary: Vibrant Coral (Energy, Action, Streaks)

### Background: Soft Cloud (Not harsh white, easier on eyes)
